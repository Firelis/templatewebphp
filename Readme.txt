================================================================

Estos archivos son una plantilla para construir un sitio web
de forma mas normalizada y por bloques usando el <?php include> 

================================================================

Estructura:

(assets/part)
	*head.php cabecera <style> lib
	*menu.php navbar, logo
	*body.php estructura generica de un body.
	*footer.php pie de de pagina
	*script.php js <scripts>
(assets/css)
	*style.css stylesheet hoja de estilos css
	
(img)
	*imagenes
	
(lib)
	*animate librerias de animate, para dar efectos a los Div o Img (https://daneden.github.io/animate.css/)
	*owlcarousel librerias de carouseles simples (https://owlcarousel2.github.io/OwlCarousel2/)
	*jquery librerias de jquery 3.4
	*bootstrap libreria de bootstrap 4
	
(index.php)
	*web principal.
